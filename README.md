# 🚀 Portafolio Personal | Victor Moreno

¡Hola! Soy **Victor Moreno**, Técnico en Programación de Software y Desarrollador Full Stack Junior. Este es el repositorio de mi portafolio personal, donde muestro mis habilidades, experiencia y los proyectos destacados que he desarrollado.

---

## 🛠️ Tecnologías utilizadas en este portafolio

Este portafolio fue diseñado centrado en el rendimiento, la accesibilidad y el SEO. Está construido con:

* **[Astro](https://astro.build/)** - Framework web para crear sitios estáticos ultra rápidos.
* **[Tailwind CSS](https://tailwindcss.com/)** - Framework de CSS para un diseño ágil y responsivo.
* **[TypeScript](https://www.typescriptlang.org/)** - Tipado estricto para mayor seguridad en el código.
* **[React](https://reactjs.org/)** - Para la integración de ciertos componentes dinámicos.

## 💻 Proyectos Destacados

Dentro del portafolio expongo aplicaciones que reflejan mis conocimientos tanto en el Frontend como en el Backend:

1. **CrediYa - Sistema de Gestión de Préstamos y Pagos**
   * *Stack:* Java, JDBC.
   * *Descripción:* Aplicación con arquitectura en capas (Dominio, Datos, Servicio y Vista) para la administración integral de créditos, cálculo de cuotas y validación de reglas de negocio.

2. **Construrrenta - Plataforma de Renta de Equipos**
   * *Stack:* Java, Spring Boot, React, TypeScript, JWT.
   * *Descripción:* Sistema integral Full Stack con Arquitectura Hexagonal, autenticación segura y gestión de roles (Administrador, Proveedor, Cliente).

3. **Gestor de Reservas de Salas de Coworking**
   * *Stack:* Python, FastAPI.
   * *Descripción:* API REST para la administración de salas y reservas, con validación de horarios y sistema de registro de usuarios.

4. **Tienda Online (E-commerce Vanilla)**
   * *Stack:* HTML, CSS, JavaScript.
   * *Descripción:* Consumo de API pública (FakeStoreAPI), renderizado dinámico del DOM y persistencia de carrito de compras usando `localStorage`.
